<h1>RootChess</h1>
Здесь в ближайшем времени появится инструкция <br>
*RootChess* - это шахматный движок с открытым исходным кодом, написанный на Python с использованием Pygame. Проект создан в рамках индивидуального школьного проекта по информатике.

## Особенности
- Уникальные пиксельные фигуры
- Встроенные темы
- История всех ходов на боковой панели
- Код полностью открыт для изучения
